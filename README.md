# 掼蛋比赛对战排表生成器
适用于多轮不淘汰积分制的掼蛋比赛
# 使用方式

0.下载全部文件（包括字体，因为我也不知道为啥matplotlib生成的图片显示中文有问题，所以干脆把字体也一起打包了...）

1.在模板xlsx文件[分组列表模板（自行填入组数及人员）.xls]中写入组数及姓名

2.运行py文件

3.在弹出的窗口中输入轮数（代码会自动检测模板文件中的组数）

4.生成xlsx的汇总表格及每轮对战图片
